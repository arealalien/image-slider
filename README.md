# image-slider
Simple and responsive image slider regardless of viewport etc.

## How does it work?
All you need to do is copy and paste the code from
```html
<div class="slide-image">
```
to
```html
</div>
```
like so:
```html
<div class="slide-image">
  <div class="title">
    <h1>Title</h1>
    <p>Undertitle</p>
  </div>
  <div class="overlay"></div>
  <img src="img.jpg" alt="" />
</div>
```
